# 🛍️ ZØYA — Premium E-Commerce Experience

متجر إلكتروني متكامل وعالي الأداء، مُصمَّم ومُطوَّر خصيصاً لعلامة **ZØYA**. يربط بين تجربة شراء سريعة وسلسة على الموبايل، ولوحة تحكم داخلية لإدارة العمليات والأرباح — بدون الاعتماد على منصات جاهزة.

مشروع **Production-Ready**: واجهة متجر، CMS للمحتوى، قاعدة بيانات للطلبات، بريد تلقائي، وتحليلات مالية — بمعمارية نظيفة وجاهزة للتوسع.

---

## 💎 الفكرة

ثلاثة محاور تعمل معاً:

1. **واجهة المتجر (Storefront)** — تجربة بصرية فاخرة (Dark / Light)، أنيميشنز تفاعلية، ومسار شراء مبسّط ومُحسَّن للموبايل.
2. **إدارة المحتوى (Sanity CMS)** — فصل المحتوى عن الكود: منتجات، تصنيفات، Reels، شحن، وشهادات العملاء دون تعديل الكود.
3. **لوحة التحكم والتحليلات** — إدارة الطلبات والمخزون والنشرات البريدية، مع نظام مالي لتتبع الأرباح والفترات الشهرية.

---

## ⚡ أبرز الميزات

### واجهة المتجر
- Homepage بأقسام: Hero، Featured Drop، منتجات، Collections، About، Testimonials، Reels، Contact
- كتالوج منتجات مع فلترة، صفحة منتج، **Quick View** وبحث من الـ Navbar
- سلة + Checkout (عنوان، شحن، كود خصم، InstaPay ورفع إثبات الدفع)
- تتبع الطلب وتعديله ضمن القواعد المسموحة
- وضع داكن / فاتح، Intro animation، صفحات قانونية، اشتراك Newsletter

### لوحة الإدارة
- دورة حياة الطلب: من الانتظار حتى التسليم أو الإلغاء
- أكواد خصم، تنبيهات مخزون منخفض ومشاكل تجهيز
- **Profit Analytics**: فترات مالية نشطة ومغلقة، أرشيف شهري، معادلة ربح موحّدة على السيرفر
- نشرات بريدية (Newsletter / Drop emails)
- مزامنة اختيارية مع Google Sheets
- حماية بوابة دخول للوحة الإدارة وـ Studio

### الأداء (PageSpeed)
- تحميل ديناميكي للمكوّنات الثقيلة (مودالات، Swiper، Reels، بحث)
- تحميل الأقسام عند الظهور في الشاشة
- صور AVIF/WebP، خطوط محسّنة، استقرار بصري (Zero CLS)
- استهداف **95%+** في Core Web Vitals على الموبايل

### الأمان وتجربة الاستخدام
- العمليات الحساسة على السيرفر فقط — لا تلاعب من المتصفح
- تباين ألوان متوافق مع معايير الوصول، وأهداف لمس **44×44px**
- التقارير والفترات المالية حسب **توقيت القاهرة**

## 🛠️ التقنيات المستخدمة

| الطبقة | التقنية |
|--------|---------|
| Framework | Next.js 16 (App Router), React 19 |
| CMS | Sanity v5 |
| Database | Supabase (PostgreSQL + Storage) |
| Styling | Tailwind CSS v4 |
| Animation | Framer Motion |
| Email | Resend |
| Rate limiting | Upstash Redis |
| Carousel | Swiper 12 |
