## ✨ الـ "Wow" Features (اللمسات الفنية)

* **Cinematic Scrolling Gallery:** نظام عرض صور مستوحى من براندات عالمية (Zara/Apple)، حيث تنساب الصور عمودياً بينما تظل بيانات المنتج ثابتة في مكانها (Sticky Info).
* **Dynamic Color System:** تبديل فوري لمحتوى الصفحة بالكامل (الصور، الألوان، الـ Badges) بناءً على اللون المختار، مع تأثيرات Fade سينمائية.
* **Micro-interactions:** * زر إضافة للعربة (Add to Bag) تفاعلي يتغير حالته مع أنيميشن تأكيدي.
    * مؤشر تقدم (Scroll Progress Bar) يتفاعل مع حركة المستخدم.
    * تأثيرات Hover Zoom متطورة للصور تعطي إحساساً بملمس القماش.
* **Purchase Engine:** نظام متكامل لاختيار المقاس (Size)، الكمية (Quantity)، وحساب السعر اللحظي.
* **Fully Responsive & Dark Mode Ready:** تصميم مرن جداً يبدو مذهلاً على الهواتف كما هو على الشاشات العملاقة.

---

## 🛠️ Stack (التقنيات المستخدمة)

- **Framework:** [Next.js 14](https://nextjs.org/) (App Router)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Context Management:** Custom Shopping Cart Context.
